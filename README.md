# Ecommerce Jewellery Website 💎

A modern and responsive e-commerce jewellery website built using React.js. The application allows users to browse jewellery products, view product details, manage shopping carts, and complete the checkout process through an intuitive and user-friendly interface.

## 🚀 Features

* Product Listing Page
* Product Details View
* User Authentication (Login / Register)
* Shopping Cart Management
* Checkout Page
* Responsive Design
* Dynamic Product Rendering
* State Management using React Context API

## 🛠️ Tech Stack

* React.js
* JavaScript (ES6)
* HTML5
* CSS3
* Vite
* Context API

## ▶️ Getting Started

1. Clone the repository:

```bash
git clone <repository-url>
```

2. Install dependencies:

```bash
npm install
```

3. Run the development server:

```bash
npm run dev
```
